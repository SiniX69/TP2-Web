Mémento HTML5 - Balises de texte

<h1> à <h6>

Balises de titres, de la plus importante (h1) à la moins importante (h6).

Exemple :

Titre principal

Sous-titre

<p>

Balise paragraphe, utilisée pour le texte courant.

Exemple : Ceci est un paragraphe.

<strong>

Met en valeur un texte important (gras).

Exemple : Texte important

<em>

Met en emphase un texte (italique).

Exemple : Texte en emphase

<u>

Souligner un texte.

Exemple : Texte souligné

<br>

Retour à la ligne simple.
Voici une nouvelle ligne.

<ul> et <li>

Liste non ordonnée.

  • HTML
  • CSS
  • JavaScript

<ol> et <li>

Liste ordonnée.

  1. Étape 1
  2. Étape 2
  3. Étape 3

<blockquote>

Bloc de citation.

Le développement de PostgreSQL respecte le standard.

<code>

Afficher du code.

Exemple : git commit -m "message"
