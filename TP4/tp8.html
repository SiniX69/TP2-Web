TP8 : Les Tableaux

Fonctions utilitaires pour manipuler un tableau de taille fixe

Introduction

Dans ce TP, il est demandé d'écrire des fonctions utilitaires permettant de travailler avec les tableaux. Ces fonctions seront réutilisables dans d'autres TP.

Le tableau contient N éléments. Il est interdit d’ajouter, d’insérer ou de supprimer une case.

Déclaration du tableau :

let N = parseInt(prompt("Combien de cases pour le tableau ?"));
let A = new Array(N);
let nbElemRempli = 0;

Au départ, le tableau est vide : toutes les cases contiennent undefined.

Organisation du travail

  1. Boîte noire : réfléchir aux entrées et aux sorties
  2. Coder la fonction dans le fichier .js
  3. Ajouter le mode d’emploi (commentaires)
  4. Tester la fonction avec l’inspecteur du navigateur

Exercices

Exercice 1

Initialiser toutes les cases du tableau avec la valeur 0.

Exercice 2

Initialiser toutes les cases avec une valeur passée en paramètre.

Exercice 3

Remplir toutes les cases avec des valeurs aléatoires entre MIN et MAX.

Exercice 4

Remplir un certain nombre de cases à partir de nbElemRempli. Si la capacité est dépassée, on remplit jusqu’à la fin.

Exercice 5

Afficher uniquement les cases remplies.

Exercice 6

Afficher toutes les cases du tableau.

Exercice 7

Afficher les cases entre les positions DEB et FIN.

Exercice 8

Remplir la première case vide. Retourne false si impossible.

Exercice 9

Permettre à l’utilisateur de saisir lui-même des valeurs jusqu’à remplir le tableau.
